Past Releases

0.2.0

Apache Paimon Mosaic 0.2.0. Key changes:

  • MAP type support with flattened columnar storage
  • Refactored child column layout metadata for complex types
  • Deterministic BPE name encoding with expanded test coverage
  • Java native library glibc baseline fix for broader Linux compatibility

0.1.0

The first release of Apache Paimon Mosaic. Key features:

  • Mosaic columnar-bucket hybrid format core implementation
  • High-performance reader and writer with ZSTD compression
  • Arrow-compatible data exchange via C Data Interface
  • Java binding with JNI (multi-platform native libraries)
  • Python binding with FFI
  • C++ headers for native integration
